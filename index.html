TimeWise

TimeWise will consist of multiple timers that will ultimatly allow you to keep track of a certain amount of time so you can stay on track.

TimeWise will be using VisualCode Studios to create and write the code for this appplication while using different languages in the code.

  • HTML
  • CSS

TimeWise was a given idea from the start and has not been changed the whole proposal, therefore the last change to the proposal was 23rd of October.

TimeWise will idealy be a well made timer for the user as well as include other features like:

  1. 10 minute timer
  2. 5 minute timer
  3. 3 minute timer
  4. 2 minute timer
  5. 1 minute timer
  6. 30 second timer

Some features TimeWise would love to see but might not get around to would include:

  1. Random Number generator between 1-5
  2. Random Name generater from everyone in class

TimeWise will be using Basic Technical Features learned in class like:

  1. Loops
  2. Selection
  3. Sequence

For now, TimeWise will not require any Advanced Technical Features in order to operate.
